# Viral Trend Spread as a Complex System

This project models meme propagation in a social network using concepts from complexity science such as avalanche dynamics, connectivity, and self-organized criticality.

## 📌 Overview
The simulation represents users as nodes in a network. A meme spreads probabilistically through connections, generating cascades (avalanches) of varying sizes.

## ⚙️ Model Features
- Random network generation
- Probabilistic meme sharing
- Peer influence effect
- Avalanche size tracking

## 📊 Results
- Distribution of meme spread sizes
- Log-log plot indicating scale-free behavior

## 🚀 How to Run

```bash
pip install -r requirements.txt
python code/simulation.py
